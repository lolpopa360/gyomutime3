시간표 최적화 엔진

시간표 생성 요청

고급 제약 조건을 설정하여 최적화된 시간표를 자동 생성합니다. 복잡한 스케줄링 문제를 지능형 알고리즘으로 해결합니다.

시간표 제약 조건 설정

시간표 최적화를 위한 상세 제약과 파라미터를 설정해주세요.

요일별 교시 수 설정

이동수업 연속 제한

적용 학년 (중복 선택)
중학교

특정 교사 시간 제외 설정

특정 교사의 특정 시간대를 제외하시겠습니까? (예: 김선생님 월요일 1교시 제외)

최적화 성능 경고

제약 조건이 많을수록 최적해를 찾기 어려워집니다. 꼭 필요한 제약만 설정하시기 바랍니다.

데이터 입력 형식

엑셀 파일 구성

  • 교사 정보 시트: 교사명, 담당과목, 가능시간
  • 교실 정보 시트: 교실명, 수용인원, 사용가능시간
  • 과목 정보 시트: 과목명, 시수, 학급별 배정
  • 학급 정보 시트: 학급명, 학년, 학생수

최적화 알고리즘 프로세스

  1. 제약 조건 매트릭스 생성
  2. 리소스 할당 최적화 실행
  3. 충돌 검증 및 해결
  4. 최적 시간표 결과 생성

알고리즘 최적화 목표

  • 교사-교실-시간 충돌 최소화
  • 이동수업 연속성 제약 준수
  • 교사 워크로드 균형 최적화
  • 교실 활용도 극대화

처리 시간

제약 조건 복잡도에 따라 1–24시간 소요. 복잡할수록 처리 시간이 증가할 수 있습니다.
